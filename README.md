# CloudSight: AWS-Powered ML Discovery

**CloudSight** is a cloud-based application for end-to-end video processing and ML content analysis. It leverages AWS services to provide scalable, efficient, and actionable insights from multimedia data.

## Features
- Upload and store videos securely using **AWS S3**
- Transcribe video audio using **AWS Transcribe**
- Normalize and process text for analysis
- Extract key phrases and topics from video content
- Visualize processed data in an interactive dashboard
- Scalable and automated pipeline using **AWS SageMaker** and cloud services

## Tech Stack
- **Backend:** Python, Flask
- **Cloud:** AWS (S3, SageMaker, Transcribe)
- **Data Processing:** NLP techniques for key phrase extraction
- **Frontend:** Dashboard (React/HTML/CSS)
